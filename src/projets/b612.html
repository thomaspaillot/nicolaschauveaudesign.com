B612 est une famille de caractères typographiques conçue pour les interfaces numériques des cockpits d’avion. Les contraintes propres à cet environnement sont nombreuses : luminosité extrêmement variable, charge cognitive élevée pour les pilotes, types d’informations variés (cartographies, listes de données, acronymes, alertes…). Le dessin du caractère a été affiné en fonction de tests perceptifs réalisés en laboratoire, afin de maximiser les propriétés permettant de discriminer les caractères entre eux — et ainsi réduire les risques de confusion, critiques dans un contexte aéronautique.
En 2017, Airbus a accepté de publier la police avec une licence open source (licence publique Eclipse) dans le cadre du projet Polarsys. En 2018, l’Étoile de l’Industrie a été decernée au projet lors de l’Observeur du Design 2018. B612 a été publiée sur Google Fonts avec une licence Open Source (OFL) et la source a été mise sur Github.
